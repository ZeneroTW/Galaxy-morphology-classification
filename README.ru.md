# 🌌 Классификация морфологии галактик — ResNet50 + трансферное обучение

> 🇬🇧 [Read in English](README.md)

Проект по глубокому обучению для автоматической классификации морфологических типов галактик по изображениям. Используется датасет **Galaxy10 SDSS** (21 785 изображений галактик, 10 классов морфологии) и двухэтапная стратегия трансферного обучения на основе **ResNet50**, предобученной на ImageNet.

Достигнута точность **84.00%** на тестовой выборке несбалансированного 10-классового датасета.

---

## Датасет

**Galaxy10 SDSS** — открытый астрономический датасет, созданный как ML-бенчмарк на основе изображений обзора SDSS с метками от добровольцев проекта Galaxy Zoo (порог согласия >55%).

| Свойство | Значение |
|---|---|
| Всего изображений | 21 785 |
| Размер изображений | 69×69 пикс., 3 канала (g, r, i полосы SDSS) |
| Классов | 10 морфологических типов |
| Формат | HDF5 (.h5), ~210 МБ |
| Баланс | Несбалансированный (доминирует Round Smooth) |

### Классы

| № | Название |
|---|---|
| 0 | Disturbed Galaxies (нарушенные) |
| 1 | Merging Galaxies (сливающиеся) |
| 2 | Round Smooth Galaxies (круглые гладкие) |
| 3 | In-between Round Smooth (промежуточные) |
| 4 | Cigar Shaped Smooth (сигарообразные) |
| 5 | Barred Spiral Galaxies (спиральные с перемычкой) |
| 6 | Unbarred Tight Spiral (плотная спираль без перемычки) |
| 7 | Unbarred Loose Spiral (рыхлая спираль без перемычки) |
| 8 | Edge-on without Bulge (ребром без выпуклости) |
| 9 | Edge-on with Bulge (ребром с выпуклостью) |

## Архитектура

**ResNet50** с предобученными весами ImageNet и кастомной классификационной головой:

```
ResNet50 backbone (заморожен на этапе 1)
    └── fc: Linear(2048 → 256) → ReLU → Dropout(0.5) → Linear(256 → 10)
```

ResNet50 выбрана среди альтернатив (VGG16, EfficientNetB0, MobileNetV2) за оптимальный баланс глубины (50 слоёв с остаточными связями), числа параметров (25M) и точности на ImageNet (76.1% Top-1).

---

## Стратегия обучения

Двухэтапное трансферное обучение:

**Этап 1 — Обучение головы (10 эпох, LR=1e-3)**
- Весь backbone заморожен
- Обучается только кастомная голова (~0.5M параметров)
- Val Accuracy: **53.88%**

**Этап 2 — Fine-tuning layer4 (15 эпох, LR=1e-4)**
- Размораживаются layer4 (~14.3M параметров) и голова; layer1–3 остаются замороженными
- Планировщик CosineAnnealingLR
- Val Accuracy: **81.16%**
- Лучший чекпоинт сохраняется в `resnet50_best.pth`

**Финальная точность на тесте: 84.00%**

## Результаты

| Метрика | Значение |
|---|---|
| Val Accuracy — Этап 1 | 53.88% |
| Val Accuracy — Этап 2 | 81.16% |
| **Test Accuracy (финал)** | **84.00%** |
| Test Loss | ≈0.50 |

Наибольшая точность по классам: галактики ребром (класс 8, 9) и сигарообразные (класс 4) — геометрически выразительные формы.

Наибольшая путаница: Round Smooth ↔ In-between Round, Barred Spiral ↔ Unbarred Tight Spiral — морфологически схожие классы, трудные даже для человека.
